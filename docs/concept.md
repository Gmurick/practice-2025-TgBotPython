# Концепция игры

## Жанр

Инди-симулятор с элементами сатиры и повседневности.

## Основная идея

Игрок берёт на себя роль раздатчика листовок в необычном городе, наполненном странными персонажами, абсурдными ситуациями и бытовыми проблемами.

## Особенности проекта

- Ироничный юмор
- Социальная сатира
- Реалистичная городская атмосфера
- Нестандартная тематика
- Сюжетные диалоги

## Целевая аудитория

- Любители инди-игр
- Поклонники симуляторов
- Игроки, предпочитающие необычные концепции
- Молодая аудитория 16–30 лет

## Игровой цикл

1. Получение задания
2. Раздача листовок
3. Взаимодействие с NPC
4. Выполнение квестов
5. Получение награды
6. Продвижение по сюжету
