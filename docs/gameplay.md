# Игровой процесс

## Основные механики

### Передвижение

Игрок свободно перемещается по игровым локациям.

### Раздача листовок

Игрок может:
- вручать листовки прохожим;
- бросать листовки;
- взаимодействовать с реакциями NPC.

### Диалоговая система

Каждый NPC имеет:
- характер;
- уникальные реплики;
- реакции на действия игрока.

### Квестовая система

Игрок выполняет задания:
- раздать определённое количество листовок;
- найти персонажей;
- выполнить сюжетные поручения.

### Система сохранений

Прогресс игрока автоматически сохраняется.

## Локации

Планируется минимум две полноценные игровые зоны:
- улицы города;
- квартира главного героя.

## Интерфейс

В игре будет реализован:
- журнал заданий;
- инвентарь;
- меню настроек;
- система уведомлений.
